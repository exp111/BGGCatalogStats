Filter Documentation

The filter is a javascript evaluated expression running inside a plays.filter(play => ...).
This means you can run most/all things you can run with javascript.
This also means that this is a massive security vulnerability if you paste anything here, if you dont know what it does.
Variables
You should have access to the play variable, which is of the type ViewerPlay.
