About

This website was made for use with the BGG Catalog app available for Android and iOS.
The main purpose of this website is to show the user more detailed insights into their plays.
Your files are never uploaded by this site and are only processed locally.

How to use

Create custom fields

To create more detailed insights into your plays you need to store extra data for each play. You can use the custom fields feature for that. To create a custom field go into the "More" tab > Custom Fields OR into a play > "More information" tab > Create field (under Custom Fields).
Your fields have to have the same name (non case-sensitive) as the ones listed below and be assigned to the correct game (may be assigned to more than that). All fields unless other specified are "Text"/"List of values" (which are also saved as Text) fields. For plays fields you can check "Display in the list of players" to easily see and edit the fields for every game instead of going to the "More information" tab.

You need to have the following fields for the following games:

Marvel Champions

Play fields:
  • Scenario
  • Modular (or Modulars*)
  • Difficulty
Player fields:
  • Hero
  • Aspect (or Aspects*)

* As you can have multiple Modulars and Aspects I added a second Text field that can be filled with a comma seperated list of values. This will have precedence over the default field.

Too Many Bones

Play fields:
  • Tyrant
  • Difficulty
Player fields:
  • Gearloc

Export and upload your plays

For most (if not all) games this website requires you to export and upload your plays. To do this you need to go to the settings: Go to the "Home" tab and press the cog symbol in the top right.
To create a backup: Backups > Create backup > Text Only.
To access the backup: Backups > See Backups > Press the three little dots next to the backup > Share and save it to your device.
The interesting file here is the "BGGCatalog-Export_<Date>-<Time>.json". To see your stats, go to the corresponding game page here and upload that file.
